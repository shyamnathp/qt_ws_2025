Meet Shiboken 🛠️

  • Binding generator tool for creating Python bindings for C++ libraries
  • Core technology behind PySide6
  • Features on par with other popular binding generators like pybind11 and nanobind:
    • Automated binding generation
    • Type system customization
    • Smart pointer handling
    • Container conversion

Learn more about Shiboken - https://doc.qt.io/qtforpython-6/shiboken6

- Introduce Shiboken as Qt's binding generation tool - Highlight its role in PySide6 - Emphasize its automated approach to binding generation - Compare its features with pybind11 and nanobind - Provide a link to the official Shiboken documentation

The Shiboken Advantage 🎯

Traditional Approaches

  • ❌ Modify source code. (e.g., SWIG, JNI, etc.)
  • ❌ Manual binding maintenance
  • ❌ Framework-agnostic solutions

Shiboken's Way

  • ✅ Non-intrusive binding generation
  • ✅ Automated updates with Qt releases
  • ✅ Qt-specific optimizations
- JNI requires modifying C++ code with special JNI macros and types - Python/C API requires extensive boilerplate in C code - SWIG needs special directives in source files - Emphasize the importance of not modifying Qt's source code - Highlight automation benefits for large frameworks

Simplified Python Integration 🐍

Write natural Python code:


      class StringModel:
          def __init__(self):
              self._text = ""

          def add_string(self, new_string: str) -> bool:
              # Simple string manipulation
              print(f"Added: {new_string}")
              self._text = new_string
              return True

      # Register with QML
      string_model = StringModel()
      AutoQmlBridge(string_model)

Use it directly in QML:


      import QtQuick
      import QtQuick.Controls
      import backend 1.0

      Column {
          spacing: 20

          TextField {
              id: inputField
              placeholderText: "Enter text"
          }

          Button {
              text: "Add String"
              onClicked: {
                  StringModel.add_string(inputField.text)
              }
          }
      }
